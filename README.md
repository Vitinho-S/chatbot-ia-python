# 🤖 Chatbot IA com Python

Este projeto é um chatbot simples utilizando a API da OpenAI.
Projeto desenvolvido como desafio da DIO.

## 🚀 Tecnologias utilizadas
- Python
- OpenAI API

## 💡 Funcionalidades
- Recebe perguntas via terminal
- Responde utilizando inteligência artificial

## ▶️ Como usar
1. Instale as dependências:
pip install openai

2. Configure sua API Key:

Windows:
setx OPENAI_API_KEY "sua-chave-aqui"

Linux/Mac:
export OPENAI_API_KEY="sua-chave-aqui"

3. Execute o projeto:
python main.py

## ⚠️ Observação
A chave da API não está incluída por segurança.
